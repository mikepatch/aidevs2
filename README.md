
# AI_Devs 2 tasks manager

A simple framework to handle AI_Devs 2 course tasks.
## Environment Variables

To run this project, you will need to add the following environment variables to your .env file (or use the .env-template, put your keys and rename to .env)

`AIDEVS_API_KEY`

`OPENAI_API_KEY`


## Run Locally

Clone the project

```bash
  git clone https://github.com/mikepatch/aidevs2.git
```

Go to the project directory

```bash
  cd aidevs-tasks
```

Install dependencies

```bash
  npm install
```

Run the scripts (taskName = .ts from tasks/)

```bash
  npm run [taskName]
```
